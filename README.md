## Installation

- Using IDE built-in plugin system:

  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "Typst Support"</kbd> >
  <kbd>Install Plugin</kbd>

## Compatible IDEs

Works in 2025.1+ IntelliJ IDEs.
